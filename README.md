# 李流し — Rinagashi

**▶ 遊ぶならこちら → https://makoto-sanekata.github.io/rinagashi/**

指でなぞるとインクが水に滲み、指を離すと選んだ言葉が浮かんで、すっと溶けて消えます。

- 下の名前（紘代・祐子・めぐ・誠）を選んでタップすると、その人へのメッセージが浮かびます
- **最後に「パージや」**を押すと、画面の文字が崩れ落ちて、大きなハートが咲き、フィナーレの写真が現れます
- 「自動演出」ONで勝手に淡い模様が流れます
- 「洗い流す」で真っ白に戻ります
- スマホ・PCどちらでも動きます（インストール・ログイン不要）
